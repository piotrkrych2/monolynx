Instrukcja instalacji w Django

Skonfiguruj swoja aplikacje Django, zeby automatycznie raportowala bledy do Monolynx.

1. Zainstaluj SDK

SDK nie wymaga zadnych zewnetrznych zaleznosci (uzywa tylko stdlib Pythona). Wybierz sposob instalacji:

Z repozytorium git (jesli masz dostep):

pip install "monolynx-sdk @ git+https://github.com/TWOJ-ORG/monolynx.git#subdirectory=sdk"

Z lokalnej sciezki (jesli masz sklonowane repo):

pip install -e /sciezka/do/monolynx/sdk/

Kopiowanie katalogu (najprostsza opcja):

# Skopiuj katalog SDK do swojego projektu Django
cp -r /sciezka/do/monolynx/sdk/src/monolynx_sdk/ /twoj-projekt/

# Struktura po skopiowaniu:
# twoj-projekt/
#   manage.py
#   monolynx_sdk/    <-- skopiowany katalog
#   twoja_aplikacja/
#   ...

2. Dodaj middleware do settings.py

Middleware musi byc pierwszym elementem listy, zeby przechwytywal wszystkie wyjatki.

MIDDLEWARE = [
    "monolynx_sdk.middleware.MonolynxMiddleware",
    # ... reszta middleware
    "django.middleware.security.SecurityMiddleware",
    "django.contrib.sessions.middleware.SessionMiddleware",
    "django.middleware.common.CommonMiddleware",
    # ...
]

3. Skonfiguruj polaczenie w settings.py

Klucz API znajdziesz na stronie edycji projektu w dashboardzie.

Opcja A -- DSN (zalecane):

MONOLYNX_DSN = "https://<TWOJ_KLUCZ_API>@adres-serwera.pl/api/v1/events"

Opcja B -- URL + klucz osobno:

MONOLYNX_URL = "https://adres-serwera.pl/api/v1/events"
MONOLYNX_API_KEY = "<TWOJ_KLUCZ_API>"

4. Opcjonalnie: srodowisko i release

MONOLYNX_ENVIRONMENT = "production"  # domyslnie "production"
MONOLYNX_RELEASE = "1.0.0"           # opcjonalne

5. Gotowe!

Nieobsluzone wyjatki z Twojej aplikacji Django beda automatycznie raportowane do Monolynx. Bledy zobaczysz na dashboardzie w zakladce projektu.

Reczne raportowanie bledow

Mozesz tez recznie wysylac bledy i wiadomosci z dowolnego miejsca w kodzie:

import monolynx_sdk

# Przechwycenie wyjatku w bloku try/except
try:
    do_something()
except Exception as e:
    monolynx_sdk.capture_exception(e)

# Wyslanie wiadomosci tekstowej
monolynx_sdk.capture_message("Cos poszlo nie tak", level="warning")
