How to MCP

Podłącz Claude Code do Open Sentry przez MCP i zarządzaj Scrumem z poziomu terminala

1

Wygeneruj token API

Przejdź do strony Tokeny API, nadaj tokenowi nazwę (np. "Claude Code") i kliknij Wygeneruj token.

Skopiuj token od razu — nie będzie wyświetlony ponownie.

2

Dodaj serwer MCP do Claude Code

Uruchom komendę w terminalu:

claude mcp add --transport http \ --header "Authorization: Bearer TWOJ_TOKEN" \ open-sentry-scrum http://localhost:8000/mcp

Lub dodaj ręcznie do pliku .mcp.json w katalogu projektu:

{ "mcpServers": { "open-sentry-scrum": { "type": "http", "url": "http://localhost:8000/mcp", "headers": { "Authorization": "Bearer ${OPEN_SENTRY_MCP_TOKEN}" } } } }

W tym wariancie ustaw zmienną środowiskową OPEN_SENTRY_MCP_TOKEN z wartością tokenu.

3

Użyj narzędzi

Po połączeniu Claude Code ma dostęp do następujących narzędzi:

Tickety

  • list_tickets — lista ticketów
  • get_ticket — szczegóły ticketu
  • create_ticket — utwórz ticket
  • update_ticket — edytuj ticket
  • delete_ticket — usuń ticket

Sprinty

  • list_sprints — lista sprintów
  • get_sprint — szczegóły sprintu
  • create_sprint — utwórz sprint
  • start_sprint — rozpocznij sprint
  • complete_sprint — zakończ sprint

Komentarze

  • list_comments — komentarze do ticketu
  • add_comment — dodaj komentarz

Przykład użycia w Claude Code:

"Stwórz ticket 'Naprawić bug logowania' w projekcie my-app z priorytetem high" "Pokaż wszystkie tickety w sprincie" "Dodaj komentarz do ticketu #3: poprawka wdrożona na produkcję"
